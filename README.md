# Онка — AI-ассистент для онкологии

AI-помощник с двумя режимами: для врача-онколога 
и пациента. Проверяет планы лечения по актуальным 
клиническим стандартам и объясняет сложное простым языком.

## Проблема
Врачи не успевают отслеживать обновления гайдов. 
Пациенты не понимают своё лечение. С 2025 года 
рекомендации Минздрава РФ обязательны для всех врачей.

## Решение
Онка — Telegram-бот с двумя ветками:
-  Врач: аудит протокола лечения + источники
-  Пациент: объяснение + эмоциональная поддержка

## Технологии
- Telegram Bot API
- OpenAI GPT-4 + Whisper
- RAG-архитектура (векторная база знаний)
- База: RUSSCO, NCCN, ESMO, Минздрав РФ

## Как запустить
1. Найди бота в Telegram: @Alonkohelperbot
2. Напиши /start
3. Выбери кто ты — врач или пациент
4. Начни пользоваться
